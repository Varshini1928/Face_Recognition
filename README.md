# Face Recognition System 

A complete, single-file face recognition system built with OpenCV and Python. 
This system can detect, train, and recognize faces in real-time using your webcam.

## Features

- **Real-time Face Recognition** - Identify faces through webcam
- **Add New Faces** - Register new people with multiple samples
- **Confidence Scoring** - Shows recognition accuracy percentage
- **Persistent Storage** - Save/load trained models
- **Multiple User Support** - Recognize different people
- **Delete Faces** - Remove people from database
- **No Internet Required** - Runs completely offline
